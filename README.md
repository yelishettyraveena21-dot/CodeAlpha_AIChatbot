# AI Chatbot

A Java-based Artificial Intelligence Chatbot developed as part of the CodeAlpha Java Programming Internship.

## Project Description

This chatbot provides interactive communication with users through both a console interface and a graphical user interface (GUI). It uses rule-based responses and basic Natural Language Processing (NLP) techniques to answer frequently asked questions.

## Features

* Interactive chatbot conversation
* Rule-based response system
* Frequently Asked Questions (FAQ) support
* Basic NLP using text preprocessing (lowercase conversion and trimming)
* Console-based chatbot interface
* Graphical User Interface (GUI) using Java Swing
* Real-time user interaction
* Easy-to-use chat environment

## Technologies Used

* Java
* Java Swing
* Scanner Class
* Event Handling
* Conditional Statements
* String Handling

## Supported Questions

Users can ask questions such as:

* Hello
* Hi
* How are you?
* What is your name?
* What is Java?
* What is AI?
* Tell me about internship
* Thank you
